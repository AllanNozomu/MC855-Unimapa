# Backend of Unimap

## Pre requisites
- Python3.6
- Virtual environemnt for python
- pip

## Installing

Create a virtual envirnoment using python 3.6
``` bash
virtualenv -p python3.6 env
```

Install all libraries
``` bash
pip install -r requirements.txt
```

Running the server
``` bash
FLASK_APP=src.unimapa flask run
```
